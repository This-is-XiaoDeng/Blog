## 简介

这是一个 PyPi 搜索程序，一定程度上解决了 pip 不支持搜包的问题

## 依赖

1. Python3
2. rich
3. requests 

## 使用方法

```bash
git clone https://github.com/This-is-XiaoDeng/PyPiSearch.git
pip install rich requests
python3 main.py
```

> ### 温馨提示
> 1. 程序启动后根据提示输入关键词并回车即可搜索
> 2. 搜索速度由包数量和网速决定
> 3. GitHub 上还提供了 Linux x86_64 的二进制可执行程序
