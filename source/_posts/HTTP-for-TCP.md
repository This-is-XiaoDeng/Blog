## 使用方法

### 客户端

1. 克隆仓库
2. 安装依赖（`pip install -r requirememts.txt`）
3. 设置 `config.json`
4. 启动客户端
5. 使用浏览器访问开启的服务器


### 服务端

1. 克隆仓库
2. 安装依赖（`pip install -r requirememts.txt`）
