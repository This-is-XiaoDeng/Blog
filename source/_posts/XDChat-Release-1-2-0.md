## 简介

一个简单的跨平台聊天软件

## 依赖

1. Python3
2. rich

## 运行

```bash
git clone https://github.com/This-is-XiaoDeng/XDChat.git
cd XDChat
python3 -m pip install rich
python3 ./main.py
```
