简介

TBOX是由这里是小邓编写有高度DIV性工具箱，您可以在这做一个自己的工具箱插件

依赖

  • Python (>= 3.9)
  • Rich
  • PyGame

构建步骤

1、安装Python3

2、安装RICH、PYGAME

pip install rich pygame

3、下载，解压源代码

4、运行

python3 ./launch.py

漏洞反馈

如果您在使用过程中遇到了bug，那么请发送log、截图和bug说明到1744793737@qq.com.
