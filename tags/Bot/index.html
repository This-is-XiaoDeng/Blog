简介

OneDisc 是一个 Discord 平台的开源 OneBot 实现

在 Discord 上实现了 OneBot V11 / V12 标准，并做了一些拓展

理论支持连接 NoneBot2、koishi 等所有支持 OneBot V11 / V12 标准的框架
